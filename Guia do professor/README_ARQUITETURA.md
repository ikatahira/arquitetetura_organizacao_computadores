# 📚 Arquitetura e Organização de Computadores - Material Didático

## 📋 Descrição

Este conjunto de arquivos contém o material completo da disciplina **Arquitetura e Organização de Computadores** para o Curso Superior de Tecnologia em Análise e Desenvolvimento de Sistemas.

## 📁 Arquivos Incluídos

### 🏠 Arquivo Principal
- **index_arquitetura.html** - Página inicial com navegação para todas as 20 semanas

### 📖 Arquivos de Aula (20 semanas)
- `arquitetura_semana01_aula.html` até `arquitetura_semana20_aula.html`
- Cada arquivo contém:
  - Objetivos da semana
  - Conteúdo teórico detalhado
  - Atividades práticas com duração estimada

### ✏️ Arquivos de Guia do Professor (20 semanas)
- `arquitetura_semana01_guia.html` até `arquitetura_semana20_guia.html`
- Cada arquivo contém:
  - Descrição completa dos exercícios
  - Rubricas de avaliação detalhadas
  - Orientações e dicas para os alunos
  - Critérios de avaliação (Insuficiente, Básico, Proficiente, Excelente)

### 📄 Arquivo Extra
- **plano_ensino_programacao_web.html** - Plano de ensino modelo para outra disciplina

## 🗂️ Estrutura do Curso

### 📚 Módulo 1: Fundamentos (Semanas 1-5)
1. Apresentação e Fundamentos
2. Sistemas Numéricos
3. Portas Lógicas e Álgebra Booleana
4. Circuitos Combinacionais
5. Circuitos Sequenciais

### 🔧 Módulo 2: Arquitetura Básica e Arduino (Semanas 6-10)
6. Arquitetura de von Neumann
7. Introdução ao Arduino - Hardware
8. Programação Arduino - Básico
9. Sensores e Atuadores
10. Projeto Prático Arduino 1

### 💾 Módulo 3: Arquitetura Interna (Semanas 11-15)
11. CPU e Ciclo de Instrução
12. Conjunto de Instruções (ISA)
13. Hierarquia de Memória
14. Memória Cache
15. Entrada e Saída (I/O)

### 📡 Módulo 4: Avançado e ESP32 (Semanas 16-20)
16. Introdução ao ESP32
17. WiFi e IoT com ESP32
18. Bluetooth e Comunicação
19. Projeto Integrador
20. Apresentação dos Projetos

## 🎯 Características do Material

### ✅ Para os Alunos
- Objetivos claros de aprendizagem
- Conteúdo organizado e progressivo
- Atividades práticas hands-on
- Estimativa de tempo para cada atividade
- Navegação fácil entre semanas

### ✅ Para os Professores
- Rubricas de avaliação prontas
- 6 critérios de avaliação por atividade
- Orientações pedagógicas
- Material visual atrativo
- Fácil de imprimir ou compartilhar

## 🚀 Como Usar

1. **Abra o arquivo `index_arquitetura.html`** em seu navegador
2. Navegue pelos módulos e semanas
3. Clique em **"📖 Aula"** para ver o conteúdo teórico
4. Clique em **"✏️ Guia"** para ver as rubricas e orientações
5. Use os botões de navegação para avançar ou voltar
