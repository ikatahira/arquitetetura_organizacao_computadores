QUESTÃO 09 Verdadeiro / Falso pts
Analise as afirmações sobre Kernel Mode vs User Mode. Marque V para verdadeiro e F para falso:
(   )   O Kernel Mode (modo privilegiado) possui acesso total ao hardware, enquanto o User Mode tem acesso restrito — se um processo em Kernel Mode travar, pode ocorrer a Tela Azul da Morte (BSOD).
(   )   Aplicativos como navegadores (Chrome, Edge) e editores de texto executam em Kernel Mode, pois precisam de acesso direto ao hardware para desempenho máximo.
(   )   Uma syscall (chamada de sistema) é o mecanismo pelo qual um programa em User Mode solicita serviços do Kernel, como abrir arquivos ou alocar memória.
(   )   Drivers de hardware (ex: driver de placa de vídeo) executam em User Mode, por isso um bug em um driver nunca pode causar instabilidade do sistema.
